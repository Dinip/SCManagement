Associate(Nullable<Int32>) This method allows adding a partner to the club if he has no role in the club, or allows removing a partner from the club if he is. Be careful with this method, since it does not have the [Authorize] flag, but should only be called when authed. Instead, it checks if the user has id (which is only possible if he is authed), and if the value is null, it redirects to the login page with the Clubs/Id as the return url.
