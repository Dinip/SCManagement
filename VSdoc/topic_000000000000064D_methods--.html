CreateClubModalityStatistics(int) Creates statistics about modalities from a specified club. Only creates monthly stats and can be run once a day. Updates the existing stats if they exist with the new values.
CreateClubPaymentStatistics(int) Creates statistics about payments (events and memberships) from a specified club. Gets the payments received from the month of previous day and updates the existing stats if they exist or creates an new one.
CreateClubUserStatistics(int) Creates statistics about users (partners and athletes) from a specified club. Only creates monthly stats. Updates the existing stats if they exist with the new values.

CreateSystemPlansStatistics() Creates statistics about system club plans. Gets the plans that exist in the month of the previous day and and updates the existing stats if they exist or creates an new one.

GetClubPaymentDetailsStatistics(int, Nullable<Int32>, Nullable<Int32>) Gets the payment details statistics for a specified club. If year is not specified, uses current year. Returns X entries for each month corresponding to each product that was "sold" that month.
GetClubPaymentStatistics(int, Nullable<Int32>, Nullable<Int32>) Gets the payment statistics for a specified club. If year is not specified, uses current year. Returns 12 entries (1 for each month) with the respective data.
GetClubUserStatistics(int, int, Nullable<Int32>, Nullable<Int32>) Gets the user statistics for a specified club. If year is not specified, uses current year. User type must be specified (id from database). Mostly used to get the number of athletes and partners.

GetSystemPaymentStatistics(Nullable<Int32>) Get the system payment statistics by product (plan) by month
GetSystemPlansShortStatistics() Gets the system plans global statistics by product (active vs canceled)
GetSystemPlansStatistics(Nullable<Int32>) Get the system statistics by product (plan)
GetUsedAndCreatedCodes() Gets the count of used and the total of created codes Min = Used Max = Total Count
